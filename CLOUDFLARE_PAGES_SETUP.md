# Cloudflare Pages Setup (Vite + React + TypeScript + Tailwind)

Use these exact values in Cloudflare Pages:

- Framework preset: `Vite`
- Build command: `npm run build`
- Build output directory: `dist`
- Root directory: `/` (leave blank if your repo root is this project)
- Node.js version: `20`

## Environment Variables (optional but recommended)

- `NODE_VERSION=20`

## Deploy Steps

1. Push this project to GitHub.
2. In Cloudflare Dashboard, open Pages.
3. Click `Create a project` and connect your GitHub repository.
4. Set the build values exactly as listed above.
5. Deploy.
